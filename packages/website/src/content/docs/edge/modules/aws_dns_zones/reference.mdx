## Inputs

The following input variables are supported:

### domains

Description: Domain names (keys) and their configs (values) for the zones.

Type:

```hcl
map(object({
    dnssec_enabled = optional(bool, false)
  }))
```

Default: n/a

## Outputs

The following outputs are exported:

### record\_manager\_role\_arn

Description: The ARN of the IAM role used to manage DNS records.

### zones

Description: Zone information

## Providers

The following providers are needed by this module:

- [aws](https://registry.terraform.io/providers/hashicorp/aws/6.40.0/docs) (6.40.0)

- [pf](https://registry.terraform.io/providers/panfactum/pf/0.0.7/docs) (0.0.7)
