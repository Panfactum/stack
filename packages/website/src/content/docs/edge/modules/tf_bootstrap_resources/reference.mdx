## Inputs

The following input variables are supported:

### lock\_table

Description: The name of the dynamodb terraform lock table..

Type: `string`

Default: n/a

### state\_bucket

Description: The name of the terraform state bucket.

Type: `string`

Default: n/a

## Outputs

No outputs.

## Providers

The following providers are needed by this module:

- [aws](https://registry.terraform.io/providers/hashicorp/aws/6.38.0/docs) (6.38.0)

- [pf](https://registry.terraform.io/providers/panfactum/pf/0.0.7/docs) (0.0.7)

- [random](https://registry.terraform.io/providers/hashicorp/random/3.8.1/docs) (3.8.1)
