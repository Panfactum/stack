# User-specific Environment Values

The following values can be used in the `.env` file of the infrastructure repo to control interactions with the Panfactum stack:

| Value                 | Required | Description                                                                                                                                                                                          |
|-----------------------|:--------:|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `LOCAL_DEV_NAMESPACE` |    X     | A short name code that should be unique to you. Used for spinning up personal development stacks in the development cluster.                                                                         |
| `GIT_USERNAME`        |    X     | The git username used for downloading private first-party IaC modules. See [instructions.](/stable-25-04/guides/iac/first-party-modules#using-a-private-git-repository)                                 |
| `GIT_PASSWORD`        |    X     | The git password used for downloading private first-party IaC modules. See [instructions.](/stable-25-04/guides/iac/first-party-modules#using-a-private-git-repository)                                 |
| `VAULT_ADDR`          |          | The HTTP(S) URL of the [Vault](https://www.vaultproject.io/) instance to connect to when authenticating with workloads running in a cluster.                                                         |
| `VAULT_TOKEN`         |          | The [Vault authentication token](https://developer.hashicorp.com/vault/docs/concepts/tokens) to use when communicating with Vault. You do not need to set this unless recovering using a root token. |
| `AUTHENTIK_TOKEN`     |          | The [Authentik](https://goauthentik.io/) API token to use for Authentik IaC Provider. Required if deploying modules that configure Authentik.                                                        |
