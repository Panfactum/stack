## Inputs

The following input variables are supported:

### config

Description: The top-level keys are the panfactum.com/pvc-group label values and the values are the corresponding labels
and annotations to apply to all PVCs in the group.

Type:

```hcl
map(object({
    labels      = optional(map(string), {})
    annotations = optional(map(string), {})
  }))
```

Default: n/a

### namespace

Description: The namespace the cluster is in

Type: `string`

Default: n/a

## Outputs

No outputs.

## Providers

The following providers are needed by this module:

- [kubectl](https://registry.terraform.io/providers/alekc/kubectl/2.1.6/docs) (2.1.6)

- [kubernetes](https://registry.terraform.io/providers/hashicorp/kubernetes/2.35.0/docs) (2.35.0)

- [pf](https://registry.terraform.io/providers/panfactum/pf/0.0.7/docs) (0.0.7)
