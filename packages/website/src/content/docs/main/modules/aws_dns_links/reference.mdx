## Inputs

The following input variables are supported:

### links

Description: Ancestor domains (keys) to their descendent zone configurations

Type:

```hcl
map(list(object({
    subdomain    = string
    name_servers = list(string)
    ds_record    = string
  })))
```

Default: n/a

## Outputs

No outputs.
