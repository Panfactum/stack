# Kubernetes GHA ARC Systems

**Source Code:** [Link](https://github.com/Panfactum/stack/tree/__currentPanfactumVersion__/packages/infrastructure/kube_gha_arc_systems)

**Type:** [Live](/docs/reference/infrastructure-modules/overview)

This module provides:

* a deployment of the [ARC](https://docs.github.com/en/actions/hosting-your-own-runners/managing-self-hosted-runners-with-actions-runner-controller/quickstart-for-actions-runner-controller) project CRDS and controllers for self-hosted
  GitHub actions
* a Moby Buildkit installation for image builds

## Providers

The following providers are needed by this module:

* [helm](https://registry.terraform.io/providers/hashicorp/helm/2.12.1/docs) (2.12.1)

* [kubernetes](https://registry.terraform.io/providers/hashicorp/kubernetes/2.27.0/docs) (2.27.0)

## Required Inputs

No required inputs.
