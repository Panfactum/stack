# Reference Values

The following values can be used in the `.env` file of the infrastructure repo to control interactions with the Panfactum stack:

| Value                 | Description                                                                                                                                                                                                |
|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `LOCAL_DEV_NAMESPACE` | A short name code that should be unique to you. Used for spinning up personal development stacks in the development cluster.                                                                               |
| `CI`                  | Should be set to `false` unless you are attempting to emulate / test the CI system.                                                                                                                      |
| `VAULT_ADDR`          | The vault instance to connect to for authentication to your system.                                                    |
