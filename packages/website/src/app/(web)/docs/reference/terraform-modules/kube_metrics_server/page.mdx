# Kubernetes Metrics Server

**Type:** [Live](./overview)

This module provides a deployment of the [metrics-server](https://github.com/kubernetes-sigs/external-dns) project.

See the [vars file](./vars.tf) for descriptions of the input parameters.

## Providers

The following providers are needed by this module:

-  [helm](https://registry.terraform.io/providers/hashicorp/helm/2.12.1/docs) (2.12.1)

-  [kubernetes](https://registry.terraform.io/providers/hashicorp/kubernetes/2.27.0/docs) (2.27.0)
