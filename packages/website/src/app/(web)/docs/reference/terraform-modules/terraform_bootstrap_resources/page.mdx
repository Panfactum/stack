# Terraform Bootstrap Resources

**Type:** [Live](./overview)

Sets up the AWS resources necessary to begin terraforming in an environment:

- s3 state bucket
- dynamodb lock table

## Providers

The following providers are needed by this module:

-  [aws](https://registry.terraform.io/providers/hashicorp/aws/5.39.1/docs) (5.39.1)

## Modules

The following Modules are called:

### <a name="module_secondary_tags"></a> [secondary\_tags](#module\_secondary\_tags)

Source: [aws_tags](./aws_tags)


### <a name="module_tags"></a> [tags](#module\_tags)

Source: [aws_tags](./aws_tags)


## Required Inputs

The following input variables are required:

### <a name="input_lock_table"></a> [lock\_table](#input\_lock\_table)

Description: The name of the dynamodb terraform lock table..

Type: `string`

### <a name="input_state_bucket"></a> [state\_bucket](#input\_state\_bucket)

Description: The name of the terraform state bucket.

Type: `string`
