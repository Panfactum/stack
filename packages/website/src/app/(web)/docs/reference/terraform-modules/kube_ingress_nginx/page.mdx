# Kubernetes Ingress Nginx

**Type:** [Live](./overview)

This module provides our standard set up network ingress control plane components in each Kubernetes clusters. It includes:

- [NGINX Ingress](https://github.com/kubernetes/ingress-nginx)
- [AWS Load Balancer Controller](https://kubernetes-sigs.github.io/aws-load-balancer-controller/v1.1/)

## Providers

The following providers are needed by this module:

-  [helm](https://registry.terraform.io/providers/hashicorp/helm/2.12.1/docs) (2.12.1)

-  [kubernetes](https://registry.terraform.io/providers/hashicorp/kubernetes/2.22.0/docs) (2.22.0)

## Required Inputs

The following input variables are required:

### <a name="input_dhparam"></a> [dhparam](#input\_dhparam)

Description: The Diffie-Hellman parameter to use for establishing perfect forward secrecy with TLS

Type: `string`

### <a name="input_environment"></a> [environment](#input\_environment)

Description: The name of the environment for the infrastructure.

Type: `string`

### <a name="input_ingress_domains"></a> [ingress\_domains](#input\_ingress\_domains)

Description: The domains that can be used for network ingress to the cluster

Type: `set(string)`

### <a name="input_is_local"></a> [is\_local](#input\_is\_local)

Type: `bool`

### <a name="input_module"></a> [module](#input\_module)

Description: The name of the module.

Type: `string`

### <a name="input_region"></a> [region](#input\_region)

Description: The region to work in.

Type: `string`

### <a name="input_version_hash"></a> [version\_hash](#input\_version\_hash)

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_enable_tls_1_2"></a> [enable\_tls\_1\_2](#input\_enable\_tls\_1\_2)

Description: Whether to enable TLS 1.2 protocols

Type: `bool`

Default: `false`

### <a name="input_ingress_timeout"></a> [ingress\_timeout](#input\_ingress\_timeout)

Description: The timeout for connections flowing through the ingress

Type: `number`

Default: `60`

### <a name="input_max_replicas"></a> [max\_replicas](#input\_max\_replicas)

Description: The maximum number of nginx-ingress replicas to deploy

Type: `number`

Default: `10`

### <a name="input_min_replicas"></a> [min\_replicas](#input\_min\_replicas)

Description: The minimum number of nginx-ingress replicas to deploy

Type: `number`

Default: `2`

### <a name="input_nginx_ingress_helm_version"></a> [nginx\_ingress\_helm\_version](#input\_nginx\_ingress\_helm\_version)

Description: The version of the nginx-ingress helm chart to deploy

Type: `string`

Default: `"4.7.1"`

### <a name="input_nginx_ingress_version"></a> [nginx\_ingress\_version](#input\_nginx\_ingress\_version)

Description: The version of nginx-ingress to deploy

Type: `string`

Default: `"v1.8.1"`

### <a name="input_vpa_enabled"></a> [vpa\_enabled](#input\_vpa\_enabled)

Description: Whether the VPA resources should be enabled

Type: `bool`

Default: `false`
