# AWS SSO Permission Bindings

**Source Code:** [Link](https://github.com/Panfactum/stack/tree/__currentPanfactumVersion__/packages/terraform/aws_account_permission_binding)

**Type:** [Internal](./overview)

Submodule to bind AWS permission sets to groups
on a per-account basis.

## Providers

The following providers are needed by this module:

* [aws](https://registry.terraform.io/providers/hashicorp/aws/5.39.1/docs) (5.39.1)

## Required Inputs

The following input variables are required:

### <a name="input_admin_groups" /> [admin\_groups](#input_admin_groups)

Description: The groups to receive aws admin aws access in this environment

Type: `list(string)`

### <a name="input_admin_permission_set_arn" /> [admin\_permission\_set\_arn](#input_admin_permission_set_arn)

Description: The arn of the admin set of permissions

Type: `string`

### <a name="input_aws_account_id" /> [aws\_account\_id](#input_aws_account_id)

Description: The AWS account ID associated with the environment

Type: `string`

### <a name="input_identity_store_id" /> [identity\_store\_id](#input_identity_store_id)

Description: The identity store ID for the AWS SSO connection

Type: `string`

### <a name="input_reader_groups" /> [reader\_groups](#input_reader_groups)

Description: The groups to receive reader aws access in this environment

Type: `list(string)`

### <a name="input_reader_permission_set_arn" /> [reader\_permission\_set\_arn](#input_reader_permission_set_arn)

Description: The arn of the reader set of permissions

Type: `string`

### <a name="input_restricted_reader_groups" /> [restricted\_reader\_groups](#input_restricted_reader_groups)

Description: The groups to receive restricted reader aws access in this environment

Type: `list(string)`

### <a name="input_restricted_reader_permission_set_arn" /> [restricted\_reader\_permission\_set\_arn](#input_restricted_reader_permission_set_arn)

Description: The arn of the restricted reader set of permissions

Type: `string`

### <a name="input_sso_instance_arn" /> [sso\_instance\_arn](#input_sso_instance_arn)

Description: The sso instance arn for the AWS SSO connection

Type: `string`

### <a name="input_superuser_groups" /> [superuser\_groups](#input_superuser_groups)

Description: The groups to receive superuser aws access in this environment

Type: `list(string)`

### <a name="input_superuser_permission_set_arn" /> [superuser\_permission\_set\_arn](#input_superuser_permission_set_arn)

Description: The arn of the superuser set of permissions

Type: `string`
