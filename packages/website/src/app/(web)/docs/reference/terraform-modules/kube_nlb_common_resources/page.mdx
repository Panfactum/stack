# NLB Common Resources

**Type:** [Utility](./overview)

Provides common resources for creating network load balancer services.

## Providers

The following providers are needed by this module:

* [aws](https://registry.terraform.io/providers/hashicorp/aws/5.39.1/docs) (5.39.1)

* [random](https://registry.terraform.io/providers/hashicorp/random/3.6.0/docs) (3.6.0)

## Required Inputs

The following input variables are required:

### <a name="input_name_prefix" /> [name\_prefix](#input_name_prefix)

Description: A prefix to the load balancer's name

Type: `string`

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_access_logs_expiration_days" /> [access\_logs\_expiration\_days](#input_access_logs_expiration_days)

Description: How long to store access logs in days.

Type: `number`

Default: `30`

### <a name="input_deregistration_delay_seconds" /> [deregistration\_delay\_seconds](#input_deregistration_delay_seconds)

Description: The deregistration delay for NLB targets in seconds

Type: `number`

Default: `300`

### <a name="input_environment" /> [environment](#input_environment)

Description: The name of the environment the infrastructure is being deployed into.

Type: `string`

Default: `null`

### <a name="input_extra_tags" /> [extra\_tags](#input_extra_tags)

Description: Extra tags or labels to add to the created resources.

Type: `map(string)`

Default: `{}`

### <a name="input_is_local" /> [is\_local](#input_is_local)

Description: Whether this module is a part of a local development deployment

Type: `bool`

Default: `false`

### <a name="input_pf_module" /> [pf\_module](#input_pf_module)

Description: The name of the Panfactum module where the containing resources are directly defined.

Type: `string`

Default: `"kube_nlb_common_resources"`

### <a name="input_pf_root_module" /> [pf\_root\_module](#input_pf_root_module)

Description: The name of the root Panfactum module in the module tree.

Type: `string`

Default: `"kube_nlb_common_resources"`

### <a name="input_region" /> [region](#input_region)

Description: The region the infrastructure is being deployed into.

Type: `string`

Default: `null`

## Outputs

The following outputs are exported:

### <a name="output_annotations" /> [annotations](#output_annotations)

Description: Annotations to add to the LoadBalancer service

## Usage

No notes
