Description: Whether this module is a part of a local development deployment

Type: `bool`

### <a name="input_kubernetes_url"></a> [kubernetes\_url](#input\_kubernetes\_url)

Description: The url to the kubernetes API server

Type: `string`

### <a name="input_module"></a> [module](#input\_module)

Description: The name of the module.

Type: `string`

### <a name="input_region"></a> [region](#input\_region)

Description: The region to work in.

Type: `string`

### <a name="input_ssh_cert_lifetime_seconds"></a> [ssh\_cert\_lifetime\_seconds](#input\_ssh\_cert\_lifetime\_seconds)

Description: The lifetime of SSH certs provisioned by Vault

Type: `number`

### <a name="input_vault_internal_url"></a> [vault\_internal\_url](#input\_vault\_internal\_url)

Description: The internal url of the vault instance

Type: `string`

### <a name="input_version_hash"></a> [version\_hash](#input\_version\_hash)

Description: The commit hash for the version. Used to reference build artifacts.

Type: `string`
