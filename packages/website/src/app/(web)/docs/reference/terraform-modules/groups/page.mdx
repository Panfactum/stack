# RBAC Groups

**Type:** [Live](./overview)

## Providers

The following providers are needed by this module:

-  [azuread](#requirement\_azuread) (~> 2.15)

## Modules

No modules.

## Required Inputs

The following input variables are required:

### <a name="input_ci_group_config"></a> [ci\_group\_config](#input\_ci\_group\_config)

Description: CI agents are assigned to these groups depending on their environment context

Type:

```hcl
map(object({
    global_admin = optional(bool, false)
  }))
```

### <a name="input_dynamic_group_config"></a> [dynamic\_group\_config](#input\_dynamic\_group\_config)

Description: Users are assigned to these groups based on their role in the organization.

Type:

```hcl
map(object({
    description   = string
    role_groups   = list(string)
    mail_nickname = optional(string)
  }))
```

### <a name="input_environment"></a> [environment](#input\_environment)

Description: The name of the environment for the infrastructure.

Type: `string`

### <a name="input_is_local"></a> [is\_local](#input\_is\_local)

Description: Whether this module is a part of a local development deployment

Type: `bool`

### <a name="input_module"></a> [module](#input\_module)

Description: The name of the module.

Type: `string`

### <a name="input_region"></a> [region](#input\_region)

Description: The region to work in.

Type: `string`

### <a name="input_role_group_config"></a> [role\_group\_config](#input\_role\_group\_config)

Description: Role groups are groups that define a user's role and access controls in the organization.

Type:

```hcl
map(object({
    description = string
    azure_roles = optional(list(string), [])
  }))
```

### <a name="input_version_hash"></a> [version\_hash](#input\_version\_hash)

Description: The commit hash for the version. Used to reference build artifacts.

Type: `string`

### <a name="input_version_tag"></a> [version\_tag](#input\_version\_tag)

Description: Name of the application version or git commit ref.

Type: `string`

## Optional Inputs

No optional inputs.

## Outputs

No outputs.
