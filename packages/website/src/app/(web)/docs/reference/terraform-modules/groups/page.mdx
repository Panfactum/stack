# RBAC Groups

**Type:** [Live](./overview)

## Providers

The following providers are needed by this module:

* [azuread](#requirement_azuread) (\~> 2.15)

## Modules

No modules.

## Required Inputs

The following input variables are required:

### <a name="input_ci_group_config" /> [ci\_group\_config](#input_ci_group_config)

Description: CI agents are assigned to these groups depending on their environment context

Type:

```hcl
map(object({
    global_admin = optional(bool, false)
  }))
```

### <a name="input_dynamic_group_config" /> [dynamic\_group\_config](#input_dynamic_group_config)

Description: Users are assigned to these groups based on their role in the organization.

Type:

```hcl
map(object({
    description   = string
    role_groups   = list(string)
    mail_nickname = optional(string)
  }))
```

### <a name="input_environment" /> [environment](#input_environment)

Description: The name of the environment for the infrastructure.

Type: `string`

### <a name="input_is_local" /> [is\_local](#input_is_local)

Description: Whether this module is a part of a local development deployment

Type: `bool`

### <a name="input_module" /> [module](#input_module)

Description: The name of the module.

Type: `string`

### <a name="input_region" /> [region](#input_region)

Description: The region to work in.

Type: `string`

### <a name="input_role_group_config" /> [role\_group\_config](#input_role_group_config)

Description: Role groups are groups that define a user's role and access controls in the organization.

Type:

```hcl
map(object({
    description = string
    azure_roles = optional(list(string), [])
  }))
```

### <a name="input_version_hash" /> [version\_hash](#input_version_hash)

Description: The commit hash for the version. Used to reference build artifacts.

Type: `string`

### <a name="input_version_tag" /> [version\_tag](#input_version_tag)

Description: Name of the application version or git commit ref.

Type: `string`

## Optional Inputs

No optional inputs.

No outputs.
