# Private S3 Bucket

**Type:** [Submodule](./overview)

This module provides our standard configuration for private S3 buckets.

Notably it includes:

- versioning
- access control
- storage tiering and expiration
- encryption
- public access blocks
- ownership controls

## Providers

The following providers are needed by this module:

-  [aws](https://registry.terraform.io/providers/hashicorp/aws/5.39.1/docs) (5.39.1)

## Modules

No modules.

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_access_policy"></a> [access\_policy](#input\_access\_policy)

Description: AWS access policy for the bucket.

Type: `string`

Default: `""`

### <a name="input_audit_log_enabled"></a> [audit\_log\_enabled](#input\_audit\_log\_enabled)

Description: Whether the bucket's audit log should be stored.

Type: `bool`

Default: `false`

### <a name="input_expire_after_days"></a> [expire\_after\_days](#input\_expire\_after\_days)

Description: Whether objects older than indicated days should be deleted. (0 to disable)

Type: `number`

Default: `0`

### <a name="input_expire_old_versions"></a> [expire\_old\_versions](#input\_expire\_old\_versions)

Description: Whether old object versions should be expired.

Type: `bool`

Default: `true`

### <a name="input_force_destroy"></a> [force\_destroy](#input\_force\_destroy)

Description: Whether the bucket can be deleted if objects still exist in it.

Type: `bool`

Default: `false`

### <a name="input_intelligent_transitions_enabled"></a> [intelligent\_transitions\_enabled](#input\_intelligent\_transitions\_enabled)

Description: Whether objects should be subject to intelligent access tiering.

Type: `bool`

Default: `true`

### <a name="input_timed_transitions_enabled"></a> [timed\_transitions\_enabled](#input\_timed\_transitions\_enabled)

Description: Whether objects should be transitioned to lower storage tiers after a certain amount of time.

Type: `bool`

Default: `false`

### <a name="input_versioning_enabled"></a> [versioning\_enabled](#input\_versioning\_enabled)

Description: Whether object versioning is enabled.

Type: `bool`

Default: `false`
