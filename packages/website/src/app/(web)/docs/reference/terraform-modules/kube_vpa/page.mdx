# Kubernetes VPA

**Type:** [Live](./overview)

This module provides a deployment of the [vertical-pod-autoscaler](https://github.com/kubernetes/autoscaler/tree/master/vertical-pod-autoscaler) project.

See the [vars file](./vars.tf) for descriptions of the input parameters.

## Providers

The following providers are needed by this module:

-  [helm](https://registry.terraform.io/providers/hashicorp/helm/2.12.1/docs) (2.12.1)

-  [kubernetes](https://registry.terraform.io/providers/hashicorp/kubernetes/2.27.0/docs) (2.27.0)

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_environment"></a> [environment](#input\_environment)

Description: The name of the environment the infrastructure is being deployed into.

Type: `string`

Default: `null`

### <a name="input_extra_tags"></a> [extra\_tags](#input\_extra\_tags)

Description: Extra tags or labels to add to the created resources.

Type: `map(string)`

Default: `{}`

### <a name="input_is_local"></a> [is\_local](#input\_is\_local)

Description: Whether this module is a part of a local development deployment

Type: `bool`

Default: `false`

### <a name="input_pf_module"></a> [pf\_module](#input\_pf\_module)

Description: The name of the Panfactum module where the containing resources are directly defined.

Type: `string`

Default: `"kube_vpa"`

### <a name="input_pf_root_module"></a> [pf\_root\_module](#input\_pf\_root\_module)

Description: The name of the root Panfactum module in the module tree.

Type: `string`

Default: `"kube_vpa"`

### <a name="input_region"></a> [region](#input\_region)

Description: The region the infrastructure is being deployed into.

Type: `string`

Default: `null`

### <a name="input_vertical_autoscaler_helm_version"></a> [vertical\_autoscaler\_helm\_version](#input\_vertical\_autoscaler\_helm\_version)

Description: The version of VPA helm chart to deploy

Type: `string`

Default: `"3.0.2"`

Description: Whether the VPA resources should be enabled
