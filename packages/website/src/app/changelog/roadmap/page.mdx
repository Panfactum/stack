# Panfactum Feature Roadmap

## Next Stable Release

The next stable release is targeted for November 2024.

You can view the intended features and status for that release [here](https://github.com/orgs/Panfactum/projects/2/views/1).

## Feature Requests

All feature requests are recorded in our [issue tracker](https://github.com/Panfactum/stack/issues). Features are prioritized
based on the following criteria:

- [Customer tier](/stack/pricing)
- Number of issue upvotes 👍
