## Providers

The following providers are needed by this module:

* [aws](https://registry.terraform.io/providers/hashicorp/aws/5.39.1/docs) (5.39.1)

## Required Inputs

The following input variables are required:

### <a name="input_ecr_repository_names" /> [ecr\_repository\_names](#input_ecr_repository_names)

Description: The names of the repositories to create.

Type: `list(string)`

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_expire_tagged_images" /> [expire\_tagged\_images](#input_expire_tagged_images)

Description: Whether tagged images should be removed after 14 days

Type: `bool`

Default: `false`

### <a name="input_is_immutable" /> [is\_immutable](#input_is_immutable)

Description: Whether immutable tags are enabled

Type: `bool`

Default: `true`

### <a name="input_trusted_account_ids" /> [trusted\_account\_ids](#input_trusted_account_ids)

Description: The ids of the accounts that have completed access to each repository.

Type: `list(string)`

Default: `[]`

### <a name="output_repos" /> [repos](#output_repos)

Description: n/a
