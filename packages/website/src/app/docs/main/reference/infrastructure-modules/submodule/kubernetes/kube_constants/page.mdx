## Outputs

The following outputs are exported:

### <a name="output_cilium_taint"></a> [cilium\_taint](#output\_cilium\_taint)

Description: The taint added to every node before the Cilium pods are ready (i.e., when networking is unavailable)

### <a name="output_cluster_important_priority_class_name"></a> [cluster\_important\_priority\_class\_name](#output\_cluster\_important\_priority\_class\_name)

Description: A Kubernetes Priority Class that is higher than the workload-important but lower than system-cluster-critical.

### <a name="output_controller_taint"></a> [controller\_taint](#output\_controller\_taint)

Description: The taint added to every EKS nodes

### <a name="output_default_priority_class_name"></a> [default\_priority\_class\_name](#output\_default\_priority\_class\_name)

Description: The default Kubernetes Priority Class

### <a name="output_panfactum_image_repository"></a> [panfactum\_image\_repository](#output\_panfactum\_image\_repository)

Description: The repository of the Panfactum devenv image with the AWS public ECR registry (public.ecr.aws).

### <a name="output_panfactum_image_tag"></a> [panfactum\_image\_tag](#output\_panfactum\_image\_tag)

Description: The tag of the Panfactum devenv image that is compatible with this module's version of the Panfactum stack.

### <a name="output_panfactum_scheduler_name"></a> [panfactum\_scheduler\_name](#output\_panfactum\_scheduler\_name)

Description: The name to use for the 'schedulerName' pod spec field when you want to use the Panfactum bin-packing pod scheduler.

### <a name="output_workload_important_priority_class_name"></a> [workload\_important\_priority\_class\_name](#output\_workload\_important\_priority\_class\_name)

Description: A Kubernetes Priority Class that is higher than the default but lower than cluster-important. Generally, all stateful systems should have this priority class.
