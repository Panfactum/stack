# Kubernetes Internal TLS Certificate

Provisions a TLS key-pair with the standard Panfactum settings.

## Providers

The following providers are needed by this module:

* [kubectl](#requirement_kubectl) (2.0.4)

* [kubernetes](https://registry.terraform.io/providers/hashicorp/kubernetes/2.27.0/docs) (2.27.0)

## Required Inputs

The following input variables are required:

### <a name="input_namespace" /> [namespace](#input_namespace)

Description: The namespace the cluster is in

Type: `string`

### <a name="input_secret_name" /> [secret\_name](#input_secret_name)

Description: The name of the secret to save the keypair to

Type: `string`

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_common_name" /> [common\_name](#input_common_name)

Description: Common name to add to add to the cert

Type: `string`

Default: `null`

### <a name="input_duration" /> [duration](#input_duration)

Description: How long the certificate will be valid for.

Type: `string`

Default: `"336h0m0s"`

### <a name="input_include_localhost" /> [include\_localhost](#input_include_localhost)

Description: Whether to include localhost in the SANs

Type: `bool`

Default: `false`

### <a name="input_include_subdomains" /> [include\_subdomains](#input_include_subdomains)

Description: Whether to include subdomains of the services via wildcard

Type: `bool`

Default: `false`

### <a name="input_is_ca" /> [is\_ca](#input_is_ca)

Description: Whether the certificate is a certificate authority certificate or not.

Type: `bool`

Default: `false`

### <a name="input_issuer_name" /> [issuer\_name](#input_issuer_name)

Description: Name of the ClusterIssuer/Issuer to use for provisioning the cert

Type: `string`

Default: `null`

### <a name="input_private_key_encoding" /> [private\_key\_encoding](#input_private_key_encoding)

Description: The encoding of the private key. Must by `PKCS1` or `PKCS8`

Type: `string`

Default: `"PKCS1"`

### <a name="input_private_key_rotation_enabled" /> [private\_key\_rotation\_enabled](#input_private_key_rotation_enabled)

Description: Whether to enable private key rotation.

Type: `bool`

Default: `true`

### <a name="input_renew_before" /> [renew\_before](#input_renew_before)

Description: How long prior to the expiration that certificate renewal will be triggered.

Type: `string`

Default: `"168h0m0s"`

### <a name="input_service_names" /> [service\_names](#input_service_names)

Description: The names of the kubernetes service(s) to use for the domain names

Type: `list(string)`

Default: `[]`

### <a name="input_usages" /> [usages](#input_usages)

Description: Usages to add to the certificate

Type: `list(string)`

Default: `[]`

### <a name="input_use_cluster_issuer" /> [use\_cluster\_issuer](#input_use_cluster_issuer)

Description: Whether to use a ClusterIssuer. If false, will use an Issuer.

Type: `bool`

Default: `true`

## Outputs

The following outputs are exported:

### <a name="output_certificate_name" /> [certificate\_name](#output_certificate_name)

Description: n/a

### <a name="output_secret_name" /> [secret\_name](#output_secret_name)

Description: n/a

No notes
