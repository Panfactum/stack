# AWS Account

<ModuleHeader name="aws_account" sourceHref="https://github.com/Panfactum/stack/tree/main/packages/infrastructure/aws_account" status="stable" type="live" />

Completes top-level configuration for each AWS account:

1. Sets up AWS account alias

## Providers

The following providers are needed by this module:

* [aws](https://registry.terraform.io/providers/hashicorp/aws/5.39.1/docs) (5.39.1)

## Required Inputs

The following input variables are required:

Description: The human-readable string for the AWS account.

Type: `string`

Description: The email address where you want to receive invoices for your organization
