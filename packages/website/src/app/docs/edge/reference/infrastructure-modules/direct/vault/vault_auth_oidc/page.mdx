# Vault OIDC Authentication

Allows SSO logins to the Vault instance via OIDC.

To be used with the [authentik\_vault\_sso](/docs/edge/reference/infrastructure-modules/authentik/authentik_vault_sso) module.

## Providers

The following providers are needed by this module:

* [vault](https://registry.terraform.io/providers/hashicorp/vault/3.25.0/docs) (3.25.0)

## Required Inputs

The following input variables are required:

### <a name="input_client_id" /> [client\_id](#input_client_id)

Description: The client id for the OIDC authentication

Type: `string`

### <a name="input_client_secret" /> [client\_secret](#input_client_secret)

Description: The client secret for the OIDC authentication

Type: `string`

### <a name="input_oidc_discovery_url" /> [oidc\_discovery\_url](#input_oidc_discovery_url)

Description: The discover url for OIDC authentication

Type: `string`

### <a name="input_oidc_issuer" /> [oidc\_issuer](#input_oidc_issuer)

Description: The bound issuer for OIDC authentication to Vault

Type: `string`

### <a name="input_oidc_redirect_uris" /> [oidc\_redirect\_uris](#input_oidc_redirect_uris)

Description: The allowed redirect URIs for OIDC authentication

Type: `list(string)`

## Optional Inputs

The following input variables are optional (have default values):

### <a name="input_admin_groups" /> [admin\_groups](#input_admin_groups)

Description: Groups that should have read and write access to this Vault

Type: `list(string)`

Default: `[]`

### <a name="input_reader_groups" /> [reader\_groups](#input_reader_groups)

Description: Groups that should have read-only access to this Vault

Type: `list(string)`

Default: `[]`

### <a name="input_restricted_reader_groups" /> [restricted\_reader\_groups](#input_restricted_reader_groups)

Description: Groups that should have restricted read-only access to this Vault

Type: `list(string)`

Default: `[]`

### <a name="input_superuser_groups" /> [superuser\_groups](#input_superuser_groups)

Description: Groups that should have superuser access to this Vault

Type: `list(string)`

Default: `[]`

### <a name="input_token_lifetime_hours" /> [token\_lifetime\_hours](#input_token_lifetime_hours)

Description: Number of hours before generated tokens expire

Type: `number`

Default: `12`

## Outputs

No outputs.
