# packages/website/src/ Table of Contents

- `components/` - Reusable UI components organized by domain: context providers, icons, form inputs, page layouts, markdown renderers, and general UI elements.
- `content/` - Markdown and MDX documentation content organized into changelogs, docs guides, and module maturity definitions.
- `content.config.ts` - Astro content collection schema definitions that validate and type frontmatter for documentation entries.
- `extends.d.ts` - TypeScript ambient declaration file for extending module types used by the website.
- `lib/` - Shared utility functions, constants, Algolia search client, and Astro plugins used across the site.
- `pages/` - Astro page routes defining the site's URL structure including docs, blog, pricing, and autopilot sections.
- `README.md` - Documentation for the source directory's organization and conventions.
- `state/` - Client-side state management for the documentation version selector.
- `styles/` - Global CSS stylesheets and Tailwind CSS utility definitions.
